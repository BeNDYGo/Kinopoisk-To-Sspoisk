⭐*буду благодарен за звездочки*⭐

# Примеры:

на страницу добавляется кнопка **Смотреть бесплатно**, **Смотреть позже** и **Меню**

![Screenshot_1](https://github.com/BeNDYGo/Kinopoisk-To-Sspoisk/blob/main/preview/Screenshot_1.png)

- **Смотреть бесплатно** - открывает фильм для просмотра
- **Смотреть позже** - добавляет фильм в отложенное
- **Меню** - открывает меню

# ⚙️ Работа:

> ### SSpoisk
> После нажатия на кнопку **Смотреть бесплатно** открывается сайт **flcksbr** с фильмом.
> 
> Добавляется кнопка **Назад**. Она открывает фильм **обратно**
> 
> *Для более приятного просмотра кнопка **прозрачная**, если не наводить на нее мышку*
